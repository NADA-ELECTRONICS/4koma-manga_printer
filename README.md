# 4koma-manga_printer
[mbed+MA大祭り 2018@初夏の名工大](https://mbed.doorkeeper.jp/events/74869)のデモ展示品、4コマ漫画プリンタです。  
# 概要
TOKYO FLIP-FLOPさんの4コマ漫画「室外機くん」5種類をスイッチを押すごとにランダムで印字します。  
4コマ漫画のイメージbmp(横384px縦自由/1bpp)をFlashAirに保存、クレイン電子さんのAirio-BaseにAS-289R2 Thermal Printer Shieldをアドオンし4コマ漫画を印字します。
