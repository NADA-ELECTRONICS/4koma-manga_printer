# 4koma-manga_printer
4koma-manga printer

# 概要
TOKYO FLIP-FLOPさんの4コマ漫画「室外機くん」5種類をスイッチを押すごとにランダムで印字します。  
4コマ漫画のイメージbmp(1bpp)をFlashAirに保存し、  
Airio-Base(クレイン電子さん)にAS-289R2 Thermal Printer Shieldをアドオンし4コマ漫画を印字します。
